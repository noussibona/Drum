HOW TO START WITH THIS TOOL?

Before going further, please bear in mind that data collection and processing is based on Powershell script and output rendering is using a built-in secure web browser.

Prerequisites:
- Powershell version: 3 or later
- Browser: Latest version
- Secure Internet connection
- Use domain joined computer

Where to start?
1 - Click on Query button to gather data from Message Center.
2 - After the query, all data are saved in a variable and following actions are avalaible:
* Sort
* Search
* Export list to file
* Send out current displayed message
* Take or view note
